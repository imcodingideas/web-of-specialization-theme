# Sage based theme

based on version `10.x-dev`

# Requires

- php 8.0 (same as server)
- node
- nvm

# Tools

load with 

```sh
source init.sh
```

try:

```sh
cli php -v
cli composer
cli wp user list
```

# Installation

```sh
nvm use # use nvm version in .nvmrc file
yarn
yarn build
cli composer install --no-dev # install dependencies using composer and specific php version
```
